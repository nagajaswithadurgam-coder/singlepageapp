# 🌐 Glassmorphism Login Page

A modern, sleek, and fully responsive Login Page built with clean HTML5 and CSS3. This project features a premium frosted-glass effect (Glassmorphism) with vibrant background shapes that automatically adapt to any device screen size.

## ✨ Features

- **Premium UI Design:** Clean glassmorphism layout using advanced CSS filters.
- **100% Fully Responsive:** Smooth auto-adjustments for mobile screens, tablets, and desktops.
- **Cross-Browser Compatible:** Optimized code with custom prefixes specifically targeting Safari and Chrome compatibility.
- **Interactive Elements:** Smooth scale transitions (`transform: scale(0.9)`) on button clicks.

## 🛠️ Tech Stack Used

- **HTML5:** Semantic structure setup
- **CSS3:** Flexbox alignments, Linear Gradients, Media Queries, and Backdrop Filters.
